# vagrant-upp

Execute vagrant up --no-provision && vagrant sandbox on
## Installation

```
$ bundle install
$ rake build
$ vagrant plugin install pkg/vagrant-upp-0.0.xx.gem
```
## Usage

```
$ vagrant upp
```
